A funky game inspired by Wordle, but with a color twist. Guess the hex codes and challenge your brain!

### Setup

1. Clone the repo:

    `git clone https://github.com/wojnet/hexle.git`

2. Install dependencies:

    ```
    npm install       # or
    yarn install      # or
    pnpm install
    ```

3. Run locally

    ```
    npm run dev       # or
    yarn dev          # or
    pnpm dev
    ```
